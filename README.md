# Oauth2Laravel


laravel version 5.8

Seguir instrucciones de https://styde.net/como-instalar-proyectos-existentes-de-laravel/
* composer require instalará dependencias
* php artisan migrate generará la bd (tablas)
* php artisan migrate --seed (creará admin, comunas,regiones y mas)
* para crear a mas admins se pueden hacer desde el crud de la app
* los vendedores se deben crear por crud app
